MongoDB Installation on Window10 in hindi (MongoDB इंस्टालेशन हिंदी में)

नमस्कार कैसे है आप लोग उम्मीद करते हैं आप अच्छे होंगे,आज के ब्लॉग में हम जानेंगे की MongoDB का Installation कैसे करते हैं और path सेट कैसे करते हैं Window10 में चलिये शुरू करते हैं स्टेप बाय स्टेप... Aug 7, 2022.

स्टेप-(1)

सबसे पहले आपको जाना है अपने मनपसंद ब्राउज़र पर मेरा मनपसंद google chrome है इसलिए मैं यहा जाउंगा और आपको टाइप करना है MongoDB और फिर आपको enter करना है फिर आपको निचे दिए गए इमेज की तरह दिखेगा फिर आपको click करना है जहाँ Arrow दिखाया गया है!


स्टेप-(2)

इसके बाद आपके सामने एक ऐसा interface आएगा अब आपको यहां Server सेलेक्ट करना है और फिर सेलेक्ट Community Server


स्टेप-(3)

इसके बाद आपके सामने एक ऐसा interface आएगा अब आपको यहां अपना Windows Select करना है और click on Download फिर आपका file Download हो जायेगा


स्टेप-(4)

इसके बाद Downloaded File को Install करें Only Click on Downloaded File फिर आपका MongoDB Community Server आपके System में Install हो जायेगा


प्रैक्टिकल के लिए मेरा Youtube Video देखें

अब देखेंगे path setting कैसे करते हैं ...


स्टेप-(5)

आपके Windows System में MongoDB Install हो गया तो फिर आपको उस Location पे जाना है जहाँ MongoDB Install है

इसके लिए आपको अपने Computer के C-Directory में जाना है फिर programs files में वहाँ आपको MongoDB का एक Folder दिखेगा उस Folder के अंदर जाना है फिर आपको MongoDB का Server और version दिखेगा उसपे Click करना है फिर आपको MongoDB का सभी Files , Folders दिखेगा!

फिर आपको bin Folder में जाना है वहाँ आपको सभी Executable Files दिख जायेगा फिर आपको वहाँ उप्पर दिए गए path को copy करना है आप निचे देख सकते है किस path को copy करना है



स्टेप-(6)

फिर आपको अपना CMD (Command Prompt) खोलना है और टाइप करना cd और Coppied Path और Enter press करना है फिर आपको टाइप करना है mongod फिर press Enter फिर आपका MongoDB Server Start हो जायेगा

स्टेप-(7)

आप अपने Screen को Scroll करेंगे तो आपको एक Normal Error दिखेगा

data Directory C:\data\db\ not found

अब आपको घबराना नही है ये Normal Error है इसके लिए आपको अपने C-Diretory में एक new Directory Create करना है data नाम से फिर आपको incide data Diretory एक db name से Directory create करना है


अब वो जो Error आया था वो आपके Operating System में avialable हो गया अब आपको CMD में जाना है और टाइप mongod फिर आपका Server start हो जायेगा कोई Error नहीं आएगा


स्टेप-(8)

अब mongo Shell को Open करेंगे...

इसके लिए आपको फिर एक नई CMD Terminal Open करना है और टाइप करना है Cd और Coppied Path जो आपने copy किआ हुआ था फिर press Enter Key अब आपके सामने mongo Shell open हो जायेगा और आपका Mongo Community server स्टार्ट हो जाएग


स्टेप-(9)

Type show dbs और press Enter फिर आपका Default Database Name show होगा अब आप New Databse create कर सकते है उसके लिए आप निचे दिए गए link को देख सकते है

स्टेप-(10)

अब आपको दोनों Terminal को Close करना है फिर एक नई Terminal को Open करना है और टाइप करना है mongod फिर press Enter Key, अब आपको एक Error दिखेगा


स्टेप-(11)

आप चाहते है की ऐसा Error न show हो और इसको हमेशा कहि भी कभी भी use करने के लिए आपको environment variable path सेट करना होगा, उसके लिए आपने जो coppied किया था path उस path को paste करना होगा इसके लिए आपको system के Advance System Settings पे जाना है , आप इमेज में देख सकते है


स्टेप-(12)

Enter advance system settings फिर आपको Environment Variables पर click करना है


स्टेप-(13)

फिर आपको path पे Double Click करना है


स्टेप-(14)

फिर New पर Click करे


स्टेप-(15)

फिर आपके पास path Add करने का text Blink आएगा बस आपको वह Path paste कर देना है वहा और click Ok Ok Ok 3 बार


स्टेप-(16)

अब आपको New terminal open करना है फिर आपको टाइप करना है mongod और press enter फिर आपको एक new terminal ओपन करना है और टाइप करना है mongo और press enter

स्टेप-(17)

आपका mongo shell open हो जायेगा अब आप system को restart कर दे उसके बाद आपको mongod command देने की जरूरत नही पड़ेगी आप केवल mongo command दे और press enter आपका shell open हो जायेगा
